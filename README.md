## 🎯 Presentation Highlights

| Topic | Content |
|-------|---------|
| **Problem** | Global data growth (180 ZB by 2025) vs. storage limitations |
| **Solution** | DNA storage with high density, durability, and low power |
| **Challenge** | Scalability vs. accuracy trade-off in information retrieval |
| **Proposed Method** | DNAformer: DNN + Tensor-Product Codes + CPL + Safety Margin |
| **Key Results** | 3200× faster, 40% more accurate, 1.6 bits/base coding rate |
| **Impact** | Paves the way for commercial DNA storage |

## 🎯 汇报要点

| 主题 | 内容 |
|------|------|
| **背景** | 全球数据量激增（2025年达180 ZB），现有存储技术瓶颈 |
| **解决方案** | DNA存储：高密度、高耐久性、低功耗 |
| **核心挑战** | 信息检索中可扩展性与准确性的权衡 |

| **研究意义** | 为DNA存储商业化应用奠定基础 |
